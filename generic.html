WHARTON UNIVERSITY OF PENNSYLVANIA

This course is designed to impact the way you think about transforming data into better decisions. This course will show you how to set a project up for success in the first phase of the project life cycle: the project initiation phase. PROJECT MANAGEMENT: COMMUNICATIONS

This course will explore how to map out a project in the second phase of the project life cycle: the project planning phase.
